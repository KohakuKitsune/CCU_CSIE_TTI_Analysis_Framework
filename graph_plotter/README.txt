這是 plot_multi.py 的 README

環境: Python 3.10.11, 記得安裝 pandas 和 matplotlib

說明: 
(1)該程式用於繪製 TTI 折線圖表，請設定裝有目標.csv檔的目錄路徑到 dir_path，
欲儲存輸出圖片的路徑到 save_path。 

(2)該程式會在save_path給定的路徑下，以一個路線為單位創建目錄，目錄名稱是路線(編號)名稱。
繪製的圖片是該路線一個月分的TTI變化。

(3)圖片的命名方式與.csv檔的資料格式遵照github上的範例 TTI data。

注意: dir_path 必須為目錄路徑，若直接給檔案會報錯並結束程式。
